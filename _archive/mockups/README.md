# Archived mockups

These 12 `.jsx` files were loose mockup leftovers at the repo root. They mirror
component names that live in the active app under `src/components/`,
`src/screens/`, and `src/shared/`, but were never imported by the runnable code
in `src/`.

Moved here so the repo root reflects only active project files.

The canonical visual mockup of the target design is
`CivitFree Personal.html` at the repo root.

Files are kept for reference only — do not edit them and do not import them
from `src/`.
